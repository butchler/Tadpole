Tadpole Installation

Things to do before installing Tadpole:
  • Make sure your config.php file is writeable by the web server.
  • Make sure the public/ folder is also writeable.
  • Create a new empty database that will store all of Tadpole's data.
Special note for sqlite3: To use a sqlite3 database, enter 'pdo' for the database driver and 'sqlite:/path/to/your/database' for the database host, and leave all other fields blank.
